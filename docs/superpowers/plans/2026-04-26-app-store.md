# App Store Listing Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Publish MeTranslate to the Apple App Store and Google Play Store by wrapping the existing PWA using Capacitor, without rewriting the app in React Native.

**Architecture:** Capacitor wraps the existing `public/` PWA in a native WebView shell. The web app runs exactly as-is. Capacitor adds native splash screen, app icon, and makes it installable from the stores. The app points to `amatranslate.app` for all API calls.

**Tech Stack:** Capacitor v6, Node.js, Xcode (Mac required for iOS), Android Studio (for Android/Google Play), Apple Developer Account ($99/yr), Google Play Console ($25 one-time)

**Prerequisites:**
- App name finalized: **MeTranslate** ✓
- Login system done (recommended but not required)
- For iOS: a Mac computer with Xcode 15+
- For Android: any computer with Android Studio

## Task 1: Install and configure Capacitor

**Files:**
- Modify: `package.json`
- Create: `capacitor.config.ts`

- [ ] **Step 1: Install Capacitor CLI and core**

```bash
cd "C:\Users\alaza\Claode folder"
npm install @capacitor/core @capacitor/cli
```

- [ ] **Step 2: Install platform plugins**

```bash
npm install @capacitor/ios @capacitor/android
npm install @capacitor/splash-screen @capacitor/status-bar
```

- [ ] **Step 3: Create `capacitor.config.ts`**

```typescript
import { CapacitorConfig } from '@capacitor/cli';

const config: CapacitorConfig = {
  appId: 'app.amatranslate.metranslate',
  appName: 'MeTranslate',
  webDir: 'public',
  server: {
    // For production: use the live URL so API calls work
    url: 'https://amatranslate.app',
    cleartext: false,
  },
  plugins: {
    SplashScreen: {
      launchShowDuration: 1500,
      backgroundColor: '#0f1117',
      showSpinner: false,
    },
    StatusBar: {
      style: 'dark',
      backgroundColor: '#0f1117',
    },
  },
};

export default config;
```

- [ ] **Step 4: Verify Capacitor can see the project**

```bash
npx cap doctor
```

Expected output: shows Node, npm versions — no errors about missing config.

- [ ] **Step 5: Commit**

```bash
git add capacitor.config.ts package.json package-lock.json
git commit -m "feat: add Capacitor config for App Store packaging"
git push
```

## Task 4: iOS — build and publish to Apple App Store

> **Requires a Mac with Xcode 15+. Do this on a Mac.**

**Files:**
- Create: `ios/` (auto-generated by Capacitor)

- [ ] **Step 1: Add iOS platform (run on Mac)**

```bash
npx cap add ios
npx cap sync ios
```

- [ ] **Step 2: Open in Xcode**

```bash
npx cap open ios
```

- [ ] **Step 3: Configure app settings in Xcode**

In Xcode → select the `App` target → General tab:
- Display Name: `MeTranslate`
- Bundle Identifier: `app.amatranslate.metranslate`
- Version: `1.0`
- Build: `1`
- Deployment Target: iOS 15.0+

- [ ] **Step 4: Set up signing**

- Xcode → Signing & Capabilities tab
- Select your Apple Developer Team (requires $99/yr Apple Developer account)
- Check "Automatically manage signing"
- Xcode will create provisioning profiles automatically

- [ ] **Step 5: Test on a device**

- Connect iPhone via USB
- Select your device in Xcode's device picker
- Click ▶ Run
- App installs and opens — verify translation, Talk, Learn features work

- [ ] **Step 6: Archive for App Store**

- In Xcode: Product → Archive
- Wait for archive to complete → Organizer window opens
- Click "Distribute App" → App Store Connect → Upload

- [ ] **Step 7: Submit in App Store Connect**

- Go to https://appstoreconnect.apple.com
- My Apps → + → New App
- Platform: iOS
- Name: MeTranslate
- Bundle ID: app.amatranslate.metranslate
- Fill in:
  - Description (same as Play Store above)
  - Keywords: `translate,translation,AI,Arabic,Spanish,language`
  - Screenshots: required for 6.5" iPhone and 12.9" iPad
  - App Review Information: provide test account if login is enabled
- Submit for Review (Apple review takes 1–3 days)

- [ ] **Step 8: Commit iOS files**

```bash
git add ios/
git commit -m "feat: add iOS Capacitor project"
git push
```
